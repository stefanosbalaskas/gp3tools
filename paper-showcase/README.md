# gp3tools paper-only synthetic showcase

This folder contains the source files for a paper-only synthetic Gazepoint workflow showcase.

The showcase is separate from the main package examples and vignettes. It is intended as a comprehensive source document for a software-paper manuscript, not as a lightweight package example.

Files:

- `01_create_paper_synthetic_gazepoint_exports.R`: generates a larger synthetic Gazepoint-like export folder.
- `02_gp3tools_paper_showcase.Rmd`: comprehensive R Markdown workflow demonstration.
- `03_render_paper_showcase.R`: renders the paper-only R Markdown file.

Generated synthetic CSV exports, rendered HTML files, figures, and tables are intentionally not committed here. They can be regenerated locally from the scripts.

The synthetic dataset is for software demonstration only. It does not contain real participant data and should not be interpreted as empirical evidence for substantive psychological, perceptual, or marketing effects.
