# Audit British and American summary-function spellings

Audit British and American summary-function spellings

## Usage

``` r
audit_gazepoint_naming_consistency(exports = NULL)
```

## Arguments

- exports:

  Optional character vector of exported function names. When `NULL`,
  current gp3tools namespace exports are inspected.

## Value

A `"gazepoint_naming_audit"` object containing paired names and an
overall status.
