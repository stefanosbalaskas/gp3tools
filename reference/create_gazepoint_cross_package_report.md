# Create a combined gp3tools-gpbiometrics workflow report

Create a combined gp3tools-gpbiometrics workflow report

## Usage

``` r
create_gazepoint_cross_package_report(x, output_file = NULL)
```

## Arguments

- x:

  A `"gazepoint_cross_package_workflow"` object.

- output_file:

  Optional Markdown output path.

## Value

Character vector containing a compact Markdown report.
