# Write gp3tools performance benchmark tables

Write gp3tools performance benchmark tables

## Usage

``` r
write_gazepoint_performance_benchmark(
  x,
  output_dir,
  prefix = "gp3tools-performance"
)
```

## Arguments

- x:

  A `"gazepoint_performance_benchmark"` object.

- output_dir:

  Output directory.

- prefix:

  Filename prefix.

## Value

Named character vector of written files.
